Week 3 Workshop Assignment: Data Subsetting with dplyr

Author

SI Analytics

Introduction

Welcome to the Week 3 workshop assignment for the Accelerated Data Analytics Bootcamp cohort.

This week focuses on one of the most important foundations of practical data analysis:

  • selecting relevant columns
  • filtering rows
  • creating analytical subsets
  • preparing clean datasets for downstream analysis

In real-world analytics work, analysts rarely work with an entire raw dataset exactly as received. Instead, they prepare focused extracts designed for specific analytical tasks, teams, or reporting needs.

For this assignment, your responsibility is to prepare structured data subsets for someone else to use. This mirrors real-world analytical workflows where analysts prepare datasets for colleagues, reporting teams, model developers, dashboards, or decision-makers.

This assignment is designed to strengthen your practical data wrangling muscle using:

  • tidyverse
  • dplyr
  • readr

You are encouraged to work carefully, read variable names closely, and consult the data dictionary whenever needed.

Learning Objectives

By completing this assignment, you should be able to:

  • Import CSV datasets into R
  • Explore dataset structure
  • Select columns using select()
  • Filter rows using filter()
  • Rename variables using rename()
  • Use helper functions such as contains()
  • Apply logical filtering conditions
  • Export cleaned subsets to CSV files
  • Organize analytical outputs systematically

Example Using a Built-In R Dataset

Instead of the Python tips dataset, we will use the built-in mtcars dataset available in R.

Preview the Dataset

head(mtcars)

Create Example Subsets

automatic_cars <- mtcars %>%
  filter(am == 0)

manual_cars <- mtcars %>%
  filter(am == 1)

high_mpg <- mtcars %>%
  filter(mpg > 25)

Export Multiple Sheets into One Excel Workbook

write_xlsx(
  list(
    "Automatic Cars" = automatic_cars,
    "Manual Cars" = manual_cars,
    "High MPG Cars" = high_mpg
  ),
  path = "outputs/mtcars_subsets.xlsx"
)
